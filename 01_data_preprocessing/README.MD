## Задача

Выяснить, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок.


## Данные

В наличии были следующие данные о клиентах банка
- Количество детей в семье
- Общий трудовой стаж в днях
- Возраст клиента в годах
- Уровень образования
- Семейное положение
- Пол клиента
- Тип занятости
- Имел ли задолженности по возврату кредита
- Ежемесячный доход
- Цель получения кредита


## Краткий общий вывод

Да, влияет


## Используемые библиотеки
*pandas*, *pymystem3*

Предобработка данных, лемматизация, исследовательский анализ данных
